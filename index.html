HTML/CSS

On the summer of 2016, I finally start to learn web-development by chance.

I had never looking forward to learn anything about web-development recently because I don't really interested in. But for certain reason, I touched this area and I found this is going be extremely helpful. In the past, I only share my works with my friends on Qzone (A social media site that similar to Facebook). But there is one problem, I can only share the picture of my work on Qzone. When I try to share some 3D animation or Game, it will be a big challenge. For example, when I share an animation I always have to convert the file format to .gif and when I try to share the minigame I made, I can only post the link for my friends to download, which I think is really inconvenience.
So, with my personal site, I don't have to convert file format every time, and it is possible to post my game here (Even I don't know to do as now) But, building a good site is always time consumming and challenge to do....
